Vertical and horizontal form groups

You should always use .form-group--vertical or .form-group--horizontal, cause labels there have more accurate width and position.

You should always place label.form-label inside div.col-x, so that label would have more accurate width and position.

Also, no .form-group--vertical or .form-group--horizontal needed with a grid.
